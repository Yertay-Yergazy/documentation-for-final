Elimatrix

Линейная алгебра без NumPy: RREF, Gauss, rank, анализ СЛУ

Python 3 No NumPy Fractions Tests

Что это

Elimatrix — внутренняя библиотека для проекта по Python II. Она реализует базовые алгоритмы линейной алгебры с нуля (без внешних библиотек) и даёт удобные функции для работы с матрицами и системами линейных уравнений (СЛУ).

✔ RREF / REF
✔ Rank
✔ Анализ 0/1/∞ решений
✔ Random generators

Архитектура

Структура проекта (минимум):

FINAL/ elimatrix/ __init__.py utils.py gauss.py solve.py decomp.py (optional) eigen.py (optional) tests/ test_gauss_rref.py test_sle_analyze.py
  • gauss.py — RREF, rref_augmented, rank
  • solve.py — analyze_sle (0 / 1 / ∞ решений)
  • utils.py — random_matrix / random_system
  • tests/ — тесты + печать результата в терминал

Функции

  • rref(A) — приведённая ступенчатая форма матрицы
  • rref_augmented(A, b) — RREF для [A|b]
  • rank(A) — ранг матрицы
  • analyze_sle(A, b) — тип решений СЛУ: 0 / 1 / ∞
  • random_matrix(rows, cols) — генерация матриц
  • random_system(n) — генерация СЛУ

Как запустить

Тесты (из папки FINAL):

python -m tests.test_gauss_rref python -m tests.test_sle_analyze

Пример использования:

from elimatrix import random_system, analyze_sle, rref A, b = random_system(3) result = analyze_sle(A, b) print("status:", result["status"]) print("rref([A|b]):", result["rref"])
