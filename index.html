О проекте

Elimatrix — учебная библиотека линейной алгебры, в которой мы реализовали метод Гаусса и связанные задачи с нуля (без NumPy).

Примеры Алгоритм Архитектура Модули Тесты Участники

Что умеет Elimatrix

  • rref(A) — приведённая ступенчатая форма матрицы (RREF)
  • rref_augmented(A, b) — RREF для расширенной матрицы [A|b]
  • rank(A) — ранг матрицы
  • analyze_sle(A, b) — анализ СЛУ: нет / единственное / бесконечно много решений
  • random_matrix(), random_system — генераторы для тестов и демо
  • determinant() — поиск детерминанта
  • inverse() — поиск обратной матрицы
Без внешних библиотек

Быстрый старт

Запуск тестов из папки FINAL:

python -m tests.test_gauss_rref python -m tests.test_sle_analyze python -m tests.test_det_inv

Пример импорта:

from elimatrix import rref, rank, analyze_sle from elimatrix.utils import random_system

Примеры использования (ввод и вывод)

Ниже наша библиотека делает и что можно ожидать на выходе.

1) RREF для матрицы

Ввод
from elimatrix import rref A = [ [2, 4, -2], [4, 9, -3], [-2, -3, 7] ] M = rref(A) print(M)
Вывод (пример)
RREF(A): [1, 0, 0] [0, 1, 0] [0, 0, 1]

Вывод зависит от матрицы; тут пример, когда матрица обратима (полный ранг).

2) Ранг матрицы

Ввод
from elimatrix import rank A = [ [1, 2], [2, 4] ] print(rank(A))
Вывод
1

Вторая строка = 2× первая → строки зависимы → ранг 1.

3) Анализ СЛУ (0 / 1 / ∞ решений)

Нет решений
from elimatrix import analyze_sle A = [ [1, 1], [1, 1] ] b = [1, 2] res = analyze_sle(A, b) print(res["status"]) print(res["rref"])
Вывод (пример)
no solutions RREF([A|b]): [1, 1, 1] [0, 0, 1] # 0 = 1 (противоречие)
Единственное решение
A = [ [1, 2], [3, 4] ] b = [5, 11] res = analyze_sle(A, b) print(res["status"]) print(res["rankA"], res["rankAb"])
Вывод (пример)
unique solution 2 2
Бесконечно много решений
A = [ [1, 1], [2, 2] ] b = [2, 4] res = analyze_sle(A, b) print(res["status"]) print(res["rankA"], res["rankAb"])
Вывод (пример)
infinitely many solutions 1 1

4) Генерация случайных данных (для демо)

Ввод
from elimatrix.utils import random_system from elimatrix import analyze_sle A, b = random_system(3, -5, 5) res = analyze_sle(A, b) print("A =", A) print("b =", b) print("status =", res["status"])
Вывод (пример)
A = [[2, -1, 3], [4, 0, 6], [-2, 1, -3]] b = [1, 2, 0] status = infinitely many solutions

Архитектура библиотеки

Минимальная структура проекта:

FINAL/ elimatrix/ __init__.py utils.py gauss.py solve.py decomp.py (determinant, inverse) eigen.py (опционально) tests/ test_gauss_rref.py test_sle_analyze.py test_det_inv.py

__init__.py — “вход” в библиотеку: экспортирует функции, чтобы писать from elimatrix import rref.

Модули: кратко что внутри

elimatrix/gauss.py
  • rref(A) — строит RREF матрицы A
  • rref_augmented(A,b) — RREF для [A|b]
  • rank(A) — ранг через RREF
elimatrix/solve.py
  • analyze_sle(A,b) — определяет тип решений по рангам и противоречию
  • возвращает status, rankA, rankAb, rref
elimatrix/utils.py
  • random_matrix — матрица со случайными целыми
  • random_system — СЛУ Ax=b для демо/тестов
elimatrix/decomp.py
  • determinant(A) — определитель (через Гаусса/преобразования)
  • inverse(A) — обратная матрица (через расширение [A|I])

Тестирование

Тесты проверяют:

  • корректность ранга на зависимых/случайных матрицах
  • корректность структуры RREF
  • три кейса СЛУ: нет / одно / бесконечно
  • det/inverse

Запуск (из FINAL):

python -m tests.test_gauss_rref python -m tests.test_sle_analyze python -m tests.test_det_inv
